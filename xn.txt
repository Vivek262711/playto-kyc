Step 1: Open Two Terminals
You need two separate terminals — one for backend, one for frontend.

Step 2: Start the Backend (Terminal 1)
powershell
# Navigate to backend
cd C:\Users\Vivek\.gemini\antigravity\scratch\playto-kyc-pipeline\backend
# Activate the virtual environment
.\venv\Scripts\activate
# Run database migrations (first time only)
python manage.py makemigrations users kyc notifications
python manage.py migrate
# Seed the database with demo data (first time only)
python manage.py shell < seed.py
# Start the Django server
python manage.py runserver 8000
You should see: Starting development server at http://127.0.0.1:8000/

Step 3: Start the Frontend (Terminal 2)
powershell
# Navigate to frontend
cd C:\Users\Vivek\.gemini\antigravity\scratch\playto-kyc-pipeline\frontend
# Install dependencies (first time only)
npm install
# Start the Vite dev server
npm run dev
You should see: Local: http://localhost:5173/

Step 4: Open in Browser
Go to http://localhost:5173/
